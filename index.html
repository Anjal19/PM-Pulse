Frameworks

Core Model

The Alignment Triangle

Every PM decision sits at the intersection of three competing forces. Understanding this triangle is the first step to making better calls — and fewer rework cycles.

  • User What the user actually needs vs. what they say they want
  • Business Revenue, retention, and the metrics that matter
  • Eng Technical feasibility and architectural constraints

AI Strategy

Build / Buy / Integrate

The strategic decision tree for AI-era PMs. Not every problem needs you to build a model. Know when to call the API.

  • Build Unique data moat + core differentiator
  • Buy Commodity need + speed matters
  • Integrate Good-enough API + cost sensitivity
  • Fine-tune Base model + domain-specific accuracy

Metrics

The North Star Framework

One metric that captures the value your product creates for users, aligned to business outcomes. Everything else is an input metric.

  • Reflects Core user value, not just business value
  • Leads Predictive of long-term retention and growth
  • Owns A single team can measurably move it
